# Jak uruchomić projekt

Potrzebujesz dwóch terminali (okien konsoli).

### 1. Backend (Serwer Python)
W pierwszym terminalu:
```bash
cd backends/Neurohackathon-pawel
python server.py
```
*To uruchomi serwer na porcie 8000. Jeśli nie masz opaski, włączy się tryb symulacji.*

### 2. Frontend (Strona WWW)
W drugim terminalu:
```bash
cd frontend
npm run dev
```
*To uruchomi stronę. Kliknij w link, który się pojawi (zazwyczaj http://127.0.0.1:5173).*
